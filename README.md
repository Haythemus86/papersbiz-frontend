# Papers Biz Frontend

Frontend Vue 3 du portail Papers Biz pour automatiser des demarches administratives.

## Stack

- Vue 3
- Vite

## Demarrage

```bash
npm install
npm run dev
```

## Build production

```bash
npm run build
```

## Contenu initial

- page d'accueil editoriale pour le portail Papers Biz
- tunnel de qualification inspire du prototype PDF fourni
- sections dediees a la creation d'entreprise, aux modifications societaires et aux demarches particuliers
